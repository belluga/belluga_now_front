# Boora Icon Font Source

This directory stores the IcoMoon export used by the Flutter runtime icon font.

- `icomoon/BooraIcons.ttf` is the source font copied to the hashed runtime
  asset declared in `assets/fonts/boora_icons_configs/config.json`.
- `icomoon/BooraIcons.otf` is kept with the export for reference.
- `icomoon/boora_icons_icomoon.dart` is the Dart class generated by IcoMoon,
  renamed to satisfy the analyzer file-name convention.
- `icomoon/boora.icomoon.json` is the IcoMoon project metadata and glyph
  contract.

The Flutter app registers `BooraIcons` as the canonical font family and keeps
`Boora` as a legacy alias in `pubspec.yaml` for previously persisted icon
payloads. The runtime filename includes the source font hash prefix so web
deploys do not reuse immutable CDN cache entries for older font bytes.
